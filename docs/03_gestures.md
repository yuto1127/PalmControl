# ジェスチャー（モード）と基本操作

この章では、手の形に基づく「モード」と、マウス操作の基本を文章で説明します（図形による説明は使いません）。

## モード（Mouse / Scroll / None）

PalmControlは、指の伸展状態から大まかなモードを推定します。

- **Mouse（マウス）**: 人差し指と中指が伸びている状態を中心に判定します。主にカーソル移動やクリック操作の文脈で使います。
- **Scroll（スクロール）**: 人差し指・中指・薬指・小指が伸びている状態を中心に判定します。スクロール操作の文脈で使います。
- **None**: 上記に該当しない状態です。基本は操作を行いません。

実運用では、指の見え方や個人差、照明条件によりモードが揺れることがあります。必要に応じて `detection.*` や `camera.roi.*` を調整してください。

## カーソル移動（相対移動）

マウスモードでのカーソル移動は、手の絶対位置へカーソルを吸い寄せる方式ではなく、フレーム間の差分を積み分ける方式（いわゆる空中トラックパッド）です。これにより、モード遷移直後の不自然なジャンプを抑えやすくします。

## クリック姿勢（誤操作防止）

設定によっては、クリック系操作に「中指を曲げた状態」を条件に含めます。これは、カーソル移動とつまみ動作が干渉しやすい問題を分離するためです。
