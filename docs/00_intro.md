# PalmControl について

PalmControlは、カメラ映像から手の状態を推定し、マウス操作へ変換する研究用アプリケーションです。研究用途を最優先し、挙動の多くを `config/settings.yaml` のパラメータで調整できるようにしています。

## このドキュメントの読み方

`docs/` 配下のMarkdownを用途別に分割しています。GUIの「マニュアル」タブでは、ファイル名順に内容が連結表示されます。

## 想定ユーザーと注意事項

- 研究・検証用途のため、OS操作（マウス入力）を伴います。最初は必ずGUI上で **OS操作をOFF** にして挙動を確認してください。
- macOSでは、カメラ権限や入力（アクセシビリティ/入力監視）に関する権限が必要になることがあります。権限が不足していると、移動だけ通ってクリック/ドラッグが不安定になることがあります。

## 起動（例）

仮想環境を有効化して依存関係を入れたうえで、次で起動します。

```bash
python -m src.main
```
